# Arkanoid Evolution (알카노이드 에볼루션)
사이버펑크 감성으로 재탄생한 전설적인 벽돌 깨기 게임입니다. 이제 단일 모드를 넘어, 로그라이크 요소와 스테이지 클리어 등 다양한 방식으로 진화한 알카노이드를 만나보세요.

## 🕹 3가지 주요 게임 모드
- **CLASSIC**: 추억의 원조 알카노이드를 화려한 그래픽으로 즐깁니다. (점수 15배 뻥튀기와 최대 10,000개의 공 설정 가능!)
- **INFINITE (Roguelike)**: 스테이지를 클리어할 때마다 랜덤하게 제시되는 3가지 강화 모듈 중 하나를 선택하여 점점 강력해지는 무한 도전 모드입니다.
- **STAGE**: 정교하게 설계된 블록 배치와 시스템 코어를 하나씩 정복해 나가는 미션형 모드입니다.

## 🚀 게임 시스템
- **강화 시스템 (Infinite Mode)**: 
  - **PADDLE BOOST**: 패들 속도 영구 증가
  - **EXTRA LIFE**: 목숨 추가 확보
  - **BIG BALLS**: 공 크기 확대로 충돌 범위 증가 등 다양한 선택지 제공
- **아이템 시스템**: 
  - **XL**: 패들 길이 일시적 증가
  - **+2**: 패들에서 새로운 공 2개 즉시 추가 생성
  - **3x**: 현재 모든 공을 각각 3개로 복제
  - **MAX**: 패들이 화면 끝까지 확장되는 강력한 효과
- **특수 블록**:
  - **DURABLE**: 여러 번 맞춰야 파괴되는 고내구도 블록
  - **EXPLOSIVE**: 파괴 시 주변 블록을 연쇄 폭발시키는 광역 파괴 블록

## 🎮 조작 방법
### PC (Keyboard & Mouse)
- **패들 이동**: 마우스 이동 또는 키보드 방향키(`←`, `→`) / `A`, `D`
- **공 발사 / 일시정지 해제**: 마우스 클릭 또는 `Space` 바
- **일시정지**: `P` 키 / **재시작**: `R` 키

### Mobile (Touch)
- **패들 이동**: 화면 터치 및 드래그
- **공 발사 / 일시정지 해제**: 화면 터치 (탭)
- **재시작**: 하단 '다시 시작' 버튼 클릭 또는 게임 종료 후 화면 터치

## 🛠 기술 스택
- **Language**: TypeScript (ESNext Modules)
- **Graphics**: HTML5 Canvas API
- **Audio**: Web Audio API (Dynamic SFX)
- **Style**: CSS3 (Cyber-Neon Theme, Responsive Design)

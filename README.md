# Arkanoid (추억의 알카노이드)
사이버펑크 감성으로 재해석한 고전 벽돌 깨기 게임입니다. 현대적인 시각 효과와 다양한 아이템, 그리고 모바일 환경 최적화를 통해 더욱 즐거운 경험을 제공합니다.

## 🚀 주요 특징
- **다양한 게임 모드**: 기본(NORMAL), 스피드(SPEED), 하드(HARD) 모드를 지원합니다.
- **아이템 시스템**: 
  - **XL**: 패들 길이 일시적 증가
  - **3x / +2**: 공 복제 및 추가 생성
  - **MAX**: 패들이 화면 끝까지 확장되는 강력한 효과
  - **LSR (LASER)**: 패들에서 레이저를 발사하여 블록 파괴
- **특수 블록**:
  - **DURABLE**: 여러 번 맞춰야 파괴되는 내구도 블록 (분홍색)
  - **EXPLOSIVE**: 파괴 시 주변 블록을 함께 터뜨리는 폭발 블록 (주황색)
- **모바일 완벽 지원**: PC는 물론 모바일 브라우저에서도 터치 조작으로 완벽하게 플레이 가능합니다.
- **네온 그래픽 & 사운드**: 화려한 네온 스타일의 시각 효과와 Web Audio API를 활용한 실시간 효과음을 제공합니다.

## 🎮 조작 방법
### PC (Keyboard & Mouse)
- **패들 이동**: 마우스 이동 또는 키보드 방향키(`←`, `→`) / `A`, `D`
- **공 발사 / 일시정지 해제 / 레이저**: 마우스 클릭 또는 `Space` 바
- **일시정지**: `P` 키
- **재시작**: `R` 키

### Mobile (Touch)
- **패들 이동**: 화면 터치 및 드래그
- **공 발사 / 일시정지 해제 / 레이저**: 화면 터치 (탭)
- **재시작**: 하단 '다시 시작' 버튼 클릭 또는 게임 종료 후 화면 중앙 터치

## 🛠 기술 스택
- **Graphics**: HTML5 Canvas API
- **Logic**: Vanilla JavaScript (ES6+ Modules)
- **Audio**: Web Audio API
- **Styling**: CSS3 (Neon effects, Responsive design)
